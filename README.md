# Oakly® — Modern Furnica

Landing page hero for **Oakly**, a modern furniture studio.
Follow along on [Instagram @oakly_furnica](https://www.instagram.com/oakly_furnica/).

## Run it

No build step — it's a static page. Open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server
```

## Structure

- `index.html` — hero markup
- `styles.css` — layout, palette, and all CSS micro-animations
- `script.js` — cursor, magnetic hover, mouse parallax, entrance sequencing
- `fonts.css` + `assets/fonts/` — self-hosted Manrope, Caveat, and Space Mono
- `assets/hero-bg.jpg` — hero background
- `logo.svg` — Oakly wordmark
