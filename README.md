# Oakly — Modern Furnica

Premium, mobile-first marketing site for **Oakly**, a custom furniture studio in Lahore, Pakistan.

- **Live sections:** hero, marquee, signature/about, collections, craft process, client stories, Instagram, contact
- **Contact:** [+92 300 5926262](tel:+923005926262) · [WhatsApp](https://wa.me/923005926262) · [@oakly_furnica](https://www.instagram.com/oakly_furnica/)

## Stack

Zero-dependency static site — semantic HTML, hand-written CSS and vanilla JS.
Self-hosted variable fonts (Cormorant Garamond + Jost), no CDNs, no build step.

Micro-interactions include: preloader with progress counter, split-line headline reveals,
scroll-triggered image/clip reveals, animated stat counters, infinite marquee, magnetic
buttons, custom cursor (desktop), hero parallax, auto-playing swipeable testimonial
slider, fullscreen menu with staggered links, and smart hide-on-scroll header.
All animations respect `prefers-reduced-motion`.

## Run locally

```sh
python3 -m http.server 8080
# open http://localhost:8080
```
